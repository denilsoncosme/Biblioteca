# Livraria
Url inicial: http://localhost:50150/AppAngular/Home.html#!/

projeto para uma auxiliar uma livraria:
Uma livraria da cidade teve um aumento no número de seus exemplares e está com um
problema para identificar todos os livros que possui em estoque. Para ajudar a livraria foi
solicitado a você desenvolver uma aplicação web para gerenciar estes exemplares.
